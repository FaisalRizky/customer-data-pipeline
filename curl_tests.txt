# Data Pipeline Test Curl Commands

### 1. Flask Mock Server (Port 5000)
# Health Check
curl -s http://localhost:5000/api/health | jq .

# List Customers (Page 1, Limit 5)
curl -s "http://localhost:5000/api/customers?page=1&limit=5" | jq .

# Single Customer Details (using CUST001)
curl -s http://localhost:5000/api/customers/CUST001 | jq .

# 404 Example
curl -s http://localhost:5000/api/customers/NON-EXISTENT | jq .


### 2. FastAPI Pipeline Service (Port 8000)
# Health Check
curl -s http://localhost:8000/api/health | jq .

# Trigger Data Ingestion
curl -s -X POST http://localhost:8000/api/ingest | jq .

# List Ingested Customers (Page 1, Limit 5)
curl -s "http://localhost:8000/api/customers?page=1&limit=5" | jq .

# Single Customer Details (using CUST001)
curl -s http://localhost:8000/api/customers/CUST001 | jq .

# Database Cleanup (Reset state)
curl -s -X DELETE http://localhost:8000/api/cleanup | jq .

# 404 Example
curl -s http://localhost:8000/api/customers/NON-EXISTENT | jq .
